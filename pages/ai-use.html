Use of Generative AI (ChatGPT)

During the development of this website, I utilized ChatGPT, a generative AI tool, to assist me in understanding various web development concepts and troubleshooting issues. ChatGPT provided explanations, code snippets, and guidance on best practices, which helped me learn and implement features effectively.

How I Used AI

  • To understand CSS layout and responsive design concepts
  • To troubleshoot navigation and linking issues between pages
  • To understand accessibility principles and implement ARIA attributes
  • To generate code snippets and examples for learning purposes

Important Note:

While ChatGPT was a valuable resource, I ensured that all code and content were my own work. I used the AI tool as a guide and reference, but I did not copy any code directly from it. This approach allowed me to learn and grow as a developer while maintaining academic integrity.
