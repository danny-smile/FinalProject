Who We Are

Dreamy Delight Ice Cream is a project created to demonstrate web design skills learned in CIS 130. It combines branding, layout design, and accessibility practices into a realistic business website.

What I Learned

In this course, I learned how to structure websites using semantic HTML, style layouts with CSS Grid and Flexbox, and build responsive designs that adapt to different screen sizes.

Accessibility & Ethics

Accessibility is important because websites should be usable for everyone, including people with disabilities. I applied techniques like alt text, clear navigation, and readable color contrast. Ethical web design also means creating inclusive experiences and avoiding misleading content.

Our Values

Accessibility

Designing content that is usable by all users, including those using assistive technologies.

Responsiveness

Ensuring the website works across mobile, tablet, and desktop devices.

Ethical Design

Creating honest, inclusive, and user-focused digital experiences.
